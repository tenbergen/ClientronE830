SETUPDSについて

-----------------------------
 1.SETUPDS.EXEについて
-----------------------------
SETUPDS.EXEは、DS-XGをDOS環境で使用する為の設定を行う
アプリケーションです。
SETUPDS.EXEは、PC起動時にDS-XGの設定を行います。
また、DS-XGをPCにインストール後の設定変更時に使用します。
設定情報は、ds.iniに保存されます。


-----------------------------
 2.起動方法について
-----------------------------
コマンドラインに以下のように入力します。

    SETUPDS [/s][/d][/?] （/s,d,?は、オプション）

SETUPDSをオプション無しで起動すると設定ダイアログが画面に表示されます。

/sオプション ： SETUPDSは、ds.iniファイルに保存されている設定にDS-XG
				のI/O port, IRQ, DMA, ボリュームを設定し、設定内容を画面に表示
                して終了します。
                （ この時は設定を変更することは出来ません。）

/dオプション ： SETUPDSは、現在の設定情報を画面に表示します。

/?オプション ： SETUPDSは、オプションについての説明を画面に表示します。


-----------------------------
 3.操作方法について
-----------------------------
SETUPDSをオプション無しで起動すると設定ダイアログが画面に表示されます。
ここでの操作を以下に示します。


< キーボード使用時 >
[ up ][ down ] キー    ---> 項目変更および設定値変更
[ + ][ - ]     キー    ---> ボリューム値変更
[ Enter ]      キー    ---> 項目選択および設定値決定
[ Esc ]        キー    ---> キャンセル


< サウンドテストについて >
サウンドテスト項目は、DS-XGに接続されたモニタースピーカーまたは、ヘッドフォン
から正常に音が出るかチェックするための機能です。
設定を変更した時は、この機能で正常に音が出ているか確認することをお勧めいたします。
また、音が聞こえず、エラーダイアログも表示されない場合は、ボリュームが下がりすぎ
ていないか確認して下さい。
エラーダイアログが表示された場合は、「 5.エラーメッセージについて 」を参照し
再設定を行ってください。


< 終了の仕方について >
メインメニューで[ ESC ]を押した後、"EXIT"または"SAVE & EXIT"を選択してください。
"EXIT"を選択した場合、設定変更は行われません。
"SAVE & EXIT"を選択した場合は、設定変更点はds.iniに保存され設定が変更されます。
ただし、正常に音が出ない設定になっていた場合は、エラーダイアログが表示され終了する
ことが出来ませんので、再設定を行うか、"EXIT"を選択して終了してください。


-----------------------------
 4.デフォルト設定について
-----------------------------
工場出荷のデフォルトは以下のようになっています。


     Device     |  I/O  | IRQ | DMA

-----------------------------
 5.エラーメッセージについて
-----------------------------
SETUPDS.EXE エラーメッセージ一覧とその対処法を以下に示します。


"Error : This program does not run in windows DOS box."
 ---> SETUPDS.EXEはDOS BOXをサポートしていません。


"Error : Invalid option."
 ---> /s, /d, /?以外のオプションが指定されています。


"Error : Can't find YAMAHA DS-XG."
 ---> DS-XGがPCにインストールされていません。


"Error : Can't access the file ( ds.ini )."
 ---> 設定情報ファイルds.iniが存在しません。
      ds.iniがSETUPDS.EXEとおなじディレクトリにあるか確認してください。


"Can't find Sound Blaster at 0xXXX"
 ---> Sound BlasterがI/O 0xXXXにありません。
      Sound BlasterのI/Oを変更してください。


"Can't detect Interrupt at IRQ X"
 ---> Sound BlasterのIRQ Xが他のデバイスによって既に使われています。
      Sound BlasterのIRQを変更してください。


"Can't detect DMA at channel X"
 ---> Sound BlasterのDMA Xが他のデバイスによって既に使われているか、
      現在選択されているDMA MODEが使用できない環境です。
      DMA MODEがPCPCIの場合は、サイドバンドが接続されているか確認してください。
      それ以外の場合は、Sound BlasterのDMAまたは、DMA MODEを変更してください。


"Can't find FM at 0xXXX"
 ---> FMがI/O 0xXXXにありません。
      FMのI/Oを変更してください。


"Can't find XXXXX.XXX”
 ---> SETUPDS.EXEに必要なファイルXXXXX.XXXが見つかりません。
      XXXXX.XXXがSETUPDS.EXEとおなじディレクトリにあるか確認してください。


-----------------------------
 6.使用上の注意について
-----------------------------
1.SETUPDS.EXEを実行するためには、SETUPDS.EXEと同じディレクトリに以下のファイルが必要です。

    dos4gw.exe
    ds.ini      ... 設定情報保存ファイル
    ymh8.wav    ... サウンドテスト用WAVファイル
    ymh16.wav   ... サウンドテスト用WAVファイル
    ymh.mid     ... サウンドテスト用MIDIファイル
    fmmidi.exe  ... MIDIファイル再生用ソフト


2.SETUPDS.EXEは、英語版ユーティリティーです。使用する前に必ず"US"モードに切り替えてから
 お使いください。（ 日本語OSを使用の方のみ ）


3.DS-XGの使用環境によっては、Sound Blasterの項目がすべてグレーアウトされ変更不可能な
 状態になります。これは、お使いのPCでSound Blasterが使用できないこと示しています。


4.DS-XGの使用環境によっては、IRQ MODEでINTA#を選択しても、INTA# IRQがグレーアウト
 され変更不可能な状態になります。
 これは、DS-XGとDS-XG以外のデバイスで同じINTA# IRQを共有しているためです。
 DS-XGを違うスロットに差し替えてから、もう一度SETUPDS.EXEを起動してください。
 （ DS-XGをスロットへ抜き差しする場合は、必ず電源を落として下さい。）


5.DS-XGの使用環境によっては、DMA MODEがグレーアウトされ変更不可能な状態になります。
 これは、DS-XGがそのDMA MODEしかサポートしていないことを示しています。
